## Details

The Provider Digital Access (PRODA) Code Generator will generate a verification code to offer additional security when logging into a Department of Human Services (DHS) Provider Digital Access account. Users need to first create a [PRODA account](https://architecture.digital.gov.au/proda-provider-digital-access), before linking the app to it.
